# Paradox Book

Interactive paradox explorer and simulator collection from Hacker News.

## Overview

A web-based collection of interactive paradox explorations and simulations, organized as reusable components. Built with JavaScript, featuring structured paradox data and interactive visualizations.

## Contents

- `components/` -- Reusable UI components
- `data/` -- Paradox definitions and data
- `scripts/` -- Build and utility scripts
- `docs/` -- Documentation
- `paradoxes.md` -- Catalogue of paradoxes

## Getting Started

```bash
npm install
```

Open `index.html` or see `docs/` for detailed usage instructions.
