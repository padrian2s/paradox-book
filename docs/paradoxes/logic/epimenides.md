# Epimenides Paradox

<div class="paradox-meta">
  <span class="badge wikipedia">Wikipedia</span>
  <span class="points">52 points</span>
</div>

A Cretan says: "All Cretans are liars." If true, then the Cretan is lying, so it's false. If false, then some Cretans tell truth, which is consistent.
