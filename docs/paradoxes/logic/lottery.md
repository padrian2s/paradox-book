# Lottery Paradox

<div class="paradox-meta">
  <span class="badge wikipedia">Wikipedia</span>
  <span class="points">42 points</span>
</div>

It's reasonable to believe any single ticket will lose, but unreasonable to believe NO ticket will win. Yet believing each loses implies believing none wins!
