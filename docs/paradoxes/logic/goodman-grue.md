# Goodman's Paradox

<div class="paradox-meta">
  <span class="badge wikipedia">Wikipedia</span>
  <span class="points">52 points</span>

Define "grue": green if observed before 2100, blue after. All evidence for "emeralds are green" equally supports "emeralds are grue"!
