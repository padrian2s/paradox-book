# Fitch's Paradox

<div class="paradox-meta">
  <span class="badge wikipedia">Wikipedia</span>
  <span class="points">40 points</span>
</div>

If all truths are knowable, then all truths are known! The modest claim that truths CAN be known implies the radical claim that they ARE known.
